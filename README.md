
# IPass Blackjack Helper

Dit is een blackjack simulator met ingebouwde algoritme die helpen om je kanst te verhogen om te winnen.

## Simulator

- 6/5 payout blackjack
- 1 keer splitten max
- geen double down na split


## Features

- Simulatie 
- Basic strategy
- Card counting
